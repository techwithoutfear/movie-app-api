# 🎬 Movie App API

A modern movie search web application built using the TMDB API.

---

## 🚀 Features

- 🔍 Search movies instantly
- 🎬 Auto trailer preview on hover with sound
- ❤️ Add / remove favorites
- 🧠 Smart movie suggestions
- 🌗 Light & Dark mode
- 📱 Fully responsive design
- 🔥 Trending movies section
- ▶️ YouTube fallback trailer support
